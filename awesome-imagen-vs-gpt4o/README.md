## 📝 项目说明

### 对比维度

1. **图像质量**: 清晰度、细节、色彩
2. **提示词理解**: 对中文提示词的理解能力
3. **风格还原**: 对特定艺术风格的把握
4. **文字渲染**: 中英文文字的准确性
5. **创意发挥**: 对模糊描述的创意解读

### 注意事项

- 部分案例需要参考图片，使用相同的参考图进行对比
- 提示词保持完全一致，不做任何修改
- 每个模型仅生成一次，不挑选最佳结果
